# Travel Trans Logistics Company Website

Welcome to the Travel Trans project! This repository contains the source code for the Travel Trans website, which serves as an online platform for our logistics company. 

## Overview

Travel Trans specializes in providing comprehensive logistics solutions, including product transportation services and state-of-the-art warehouse facilities. Our mission is to ensure timely and efficient delivery of goods while maintaining the highest standards of service.

## Project Structure

The project is organized as follows:

- **src/**: Contains all the source files for the website.
  - **index.html**: The homepage introducing Travel Trans and its services.
  - **services.html**: Detailed descriptions of our product transportation services.
  - **facilities.html**: Information about our warehouse facilities, including capacity and features.
  - **contact.html**: Contact information and inquiry form for potential clients.
  - **css/**: Contains stylesheets for the website.
    - **styles.css**: The main stylesheet defining the layout and design.
  - **js/**: Contains JavaScript files for interactive functionality.
    - **script.js**: JavaScript for form validation and dynamic content.
  
- **assets/**: Directory for storing video assets related to the company.
  - **videos/**: Promotional videos and service demonstrations.

- **README.md**: Documentation for the project, including setup instructions and developer information.

- **package.json**: Configuration file for npm, listing dependencies and scripts.

## Setup Instructions

To get started with the Travel Trans website, follow these steps:

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install any necessary dependencies using npm (if applicable).
4. Open `src/index.html` in your web browser to view the homepage.
